This is a search engine which uses AI to find events for the Amsterdam Dance Event (ADE) in 2024.

What it can do:
  • Find events by artist name, genre, date, label, price or venue
  • Find events with music similar to a specific DJ.
  • Search for vibe (e.g. something melanchonic)
  • Search for random activities (e.g. painting, running)
What it can't do (yet):
  • Set up a schedule for ADE
  • Handle requests which are not search related
  • Complex queries
